1. To open Front-end index.html, simply run it in the folder it is located
2. To run the Node.js backend, follow these steps:
  - Change your directory to where the app.js is found. (cd /path/app.js)
  - initialize npm with "npm init"
  - use "npm install" to get needed dependencies
  - run the app with "node app.js"

All files and codes are in the "Assignment_3" folder found in: https://github.com/Fiblmao/TCSS460
